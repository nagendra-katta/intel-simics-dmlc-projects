2.2 List by Categories

Processors

<riscv-rv32>.print-page-table print page table entries
<riscv-rv32>.print-pma-regions print current PMA regions
<riscv-rv32>.print-pmp-regions print current PMP regions
<riscv-rv32ema>.print-pma-regions print current PMA regions
<riscv-rv64>.print-page-table print page table entries
<riscv-rv64>.print-pma-regions print current PMA regions
<riscv-rv64>.print-pmp-regions print current PMP regions

Profiling

<riscv-rv32>.aprof-views manipulate list of selected address profiling views
<riscv-rv32ema>.aprof-views manipulate list of selected address profiling views
<riscv-rv64>.aprof-views manipulate list of selected address profiling views
