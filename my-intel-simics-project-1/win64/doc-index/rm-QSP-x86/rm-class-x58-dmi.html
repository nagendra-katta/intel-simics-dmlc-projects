x58-dmi

Description

DMI unit in the Intel® X58 Express Chipset.

Interfaces Implemented

conf_object, log_object, pcie_device, transaction_translator, translation_flush

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.
state-change
Notifier on potential DML state change

Port Objects

bank.io_regs
x58-dmi.io_regs

port.msg
x58-dmi.msg – legacy PCI INTx message to pci_interrupt converter
port.phy
x58-dmi.phy

Commands for this class

  • info – print information about the object
  • status – print status of the object

downstream_target
Optional attribute; read/write access; type: o|[os]|n
PCIe Downstream Target

Optional attribute; read/write access; type: [o*]
List of configuration objects from which transactions will be remapped using the graphics remapping unit

Graphics DMA remapping unit

Target to receive PCIe Legacy Interrupts

Required interfaces: pci_interrupt.
