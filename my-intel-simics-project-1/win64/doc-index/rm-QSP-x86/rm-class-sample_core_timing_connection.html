sample_core_timing_connection

Description

Sample x86 core timing connection

Interfaces Implemented

conf_object, log_object, instrumentation_connection, telemetry

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.

Commands for this class

  • info – print information about the object
  • status – print status of the object

Attributes

cpu
Pseudo attribute; read-only access; type: o
Returns the connected cpu
static_events
Pseudo attribute; read-only access; type: [[si]*]
Statistics for static events, like read, write and total number of instructions.
dynamic_events
Pseudo attribute; read-only access; type: [[si]*]
Statistics for instruction classes.
telemetry_data
Pseudo attribute; read-only access; type: [[sff]*]
Telemetry for the core together with average values.
