sample_core_timing_tool

Description

Sample x86 performance tool.

Interfaces Implemented

conf_object, log_object, instrumentation_tool

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.

Commands for this class

  • add-instruction-class – add instruction class with ipc and cdyn adjustments
  • add-instrumentation – add instrumentation
  • delete – deletes instrumentation tool
  • disable-instrumentation – disable instrumentation
  • enable-instrumentation – enable instrumentation
  • info – print information about the object
  • list-instruction-classes – list instruction classes
  • list-metrics – list metrics of the simple performance model
  • list-performance-metrics – list performance metrics from the model
  • list-providers – list-provider objects
  • remove-instruction-class – remove instruction class
  • remove-instrumentation – remove instrumentation
  • set-activity-per-read – set extra activity per read operation
  • set-activity-per-write – set extra activity per write operation
  • set-background-activity-per-cycle – set background activity per cycle
  • set-base-activity-per-instruction – set activity per instruction
  • set-base-cycles-per-instruction – set base cycles per instruction
  • set-cycles-per-read – set extra cycles per read operation
  • set-cycles-per-write – set extra cycles per write operation
  • status – print status of the object

Attributes

connections
Pseudo attribute; read-only access; type: [o*]
Example attribute, returns the connections
instruction_classes
Pseudo attribute; read/write access; type: [[sff]*]
List of instruction classes. Each class consists of a string to look for, fraction of extra cycles per instance and extra activity per instance
background_activity
Pseudo attribute; read/write access; type: f
Background activity per cycle
base_cycles
Pseudo attribute; read/write access; type: f
Baseline cycles per instruction
base_activity
Pseudo attribute; read/write access; type: f
Baseline activity per instruction
read_cycles
Pseudo attribute; read/write access; type: f
Extra cycles per explicit read access
write_cycles
Pseudo attribute; read/write access; type: f
Extra cycles per explicit write access
read_activity
Pseudo attribute; read/write access; type: f
Extra activity per explicit read access
write_activity
Pseudo attribute; read/write access; type: f
Extra activity per explicit write access
