ich10_hpe_timer

Description

High precision event timer in Intel® ICH10

Interfaces Implemented

conf_object, log_object, io_memory, simple_interrupt

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.

Port Objects

bank.regs
ich10_hpe_timer.regs
port.HRESET
ich10_hpe_timer.HRESET
port.SRESET
ich10_hpe_timer.SRESET

Commands for this class

  • info – print information about the object
  • status – print status of the object

Attributes

comp_write_slow_cycles
Optional attribute; read/write access; type: i
Number of cycles the cpu should be stalled when the comparator register is written (performance optimization)
intc_8259
Optional attribute; read/write access; type: o|[os]|n
Connection to Intel 8259A Programmable Interrupt Controller.

Required interfaces: simple_interrupt.

intc_apic
Optional attribute; read/write access; type: o|[os]|n
Connection to APIC interrupt controller.

Required interfaces: ioapic.

lpc
Optional attribute; read/write access; type: o|[os]|n
LPC bridge used to deliver FSB interrupts.If no LPC bridge is specified, FSB interrupts will be sent directlyon the connected memory space.

Required interfaces: hpet_msi.

memory_space
Optional attribute; read/write access; type: o|[os]|n
Memory space used for FSB interrupt deliveryused if the LPC bridge is not used for FSB interrupt delivery.

Required interfaces: memory_space.

start_time
Optional attribute; read/write access; type: f
Latest start time of the main counter
