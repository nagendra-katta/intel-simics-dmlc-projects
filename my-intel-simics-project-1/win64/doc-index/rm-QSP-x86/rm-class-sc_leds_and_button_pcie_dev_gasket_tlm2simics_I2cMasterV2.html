sc_leds_and_button_pcie_dev_gasket_tlm2simics_I2cMasterV2

Description

Class for binding Simics interface to SystemC Gasket.

Interfaces Implemented

conf_object, log_object

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.

Attributes

initiator
Required attribute; read/write access; type: s
Name of the initiator socket the gasket is connected to.
simulation
Required attribute; read/write access; type: o
Simics object implementing the SystemC Simulation interface.
object
Optional attribute; read/write access; type: o|n
Simics object implementing the corresponding interface of this gasket.
