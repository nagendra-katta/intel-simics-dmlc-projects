sc_leds_and_button_pcie_dev_gasket_composite_Pcie

Description

Class for binding Simics interface to SystemC Pcie Gasket.

Interfaces Implemented

conf_object, log_object, pcie_device, transaction, signal

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.

Port Objects

port.io
pcie_map_helper_cpp – The help object maps to the PCIe IO space
port.mem
pcie_map_helper_cpp – The help object maps to the PCIe MEM space
port.msg
pcie_map_helper_cpp – The help object mapes to the PCIe MSG space

Attributes

device
Required attribute; read/write access; type: s
Name of the SystemC object that implements the PcieDeviceQueryInterface and PcieBaseAddressRegisterQueryInterface.All TLM2 sockets returned by the object must have BUSWIDTH = 32 and TYPES = tlm::tlm_base_protocol_types
simulation
Required attribute; read/write access; type: o
Simics object implementing the SystemC Simulation interface.
upstream_target
Optional attribute; read/write access; type: o|n
The PCIe upstream target to connect to.
address_id_mem_map
Pseudo attribute; read-only access; type: [[[ii]i]*]
Internal. The map from MEM address range to socket ID
address_id_io_map
Pseudo attribute; read-only access; type: [[[ii]i]*]
Internal. The map from IO address range to socket ID
enable_base_address_subtraction
Optional attribute; read/write access; type: b
If set, MEM/IO transaction will receive only the offset to the base address. Otherwise, it receives the whole base address + offset. Default enabled
