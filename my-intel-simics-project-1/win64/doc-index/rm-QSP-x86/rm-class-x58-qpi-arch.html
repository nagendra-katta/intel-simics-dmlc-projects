x58-qpi-arch

Description

Intel® QuickPath Architecture Non-Core and System Address Decoder Registers in the Core™ i7 processor.

Interfaces Implemented

conf_object, log_object, pcie_device

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.
state-change
Notifier on potential DML state change

Port Objects

bank.f0
x58-qpi-arch.f0 – non-core registers
bank.f1
x58-qpi-arch.f1 – system address decoder registers
port.HRESET
x58-qpi-arch.HRESET
port.mcfg
x58-qpi-arch.mcfg
port.phy
x58-qpi-arch.phy

Commands for this class

  • info – print information about the object
  • status – print status of the object

Attributes

cfg_space
Required attribute; read/write access; type: o|[os]
Downstream PCIe Config space, mapped by PCIEXBAR
