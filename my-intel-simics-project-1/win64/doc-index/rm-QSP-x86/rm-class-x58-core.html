x58-core

Description

PCIe Core Functions in the Intel® X58 Express Chipset.

Interfaces Implemented

conf_object, log_object, pcie_device

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.
state-change
Notifier on potential DML state change

Port Objects

bank.f0
x58-core.f0 – system management registers
bank.f1
x58-core.f1 – GPIO and scratchpad registers
bank.f2
x58-core.f2 – IOH system / control status registers
bank.f3
x58-core.f3 – on-die throttling registers
port.HRESET
x58-core.HRESET
port.phy
x58-core.phy
remap_space
memory-space
remap_unit[0..1]
x58-remap-unit – The DMA/Interrupt remapping unit

Commands for this class

  • info – print information about the object
  • status – print status of the object

Attributes

external_remap_unit
Optional attribute; read/write access; type: [o|[os]|n{0:2}]
External DMA/Interrupt remapping units. If set, will be used in place of the internal objects in 'remap_unit'
vtbar_size_18_bits
Optional attribute; read/write access; type: b
If TRUE, VTBAR will use 18 bits
