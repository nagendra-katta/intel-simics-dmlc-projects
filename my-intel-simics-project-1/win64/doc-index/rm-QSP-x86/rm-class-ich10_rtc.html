ich10_rtc

Description

Real time clock in Intel® ICH10.

Interfaces Implemented

conf_object, log_object, io_memory

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.

Port Objects

bank.fixed_io
ich10_rtc.fixed_io
bank.rtc
ich10_rtc.rtc
port.HRESET
ich10_rtc.HRESET
port.SRESET
ich10_rtc.SRESET

Commands for this class

  • get-date-time – get date and time
  • info – print information about the object
  • set-date-time – set date and time
  • status – print status of the object

Attributes

irq_dev
Optional attribute; read/write access; type: o|[os]|n
Connect to serial interrupt controller
irq_level
Optional attribute; read/write access; type: i
Interrupt level for RTC device
next_alarm_time
Optional attribute; read/write access; type: f
Simulation time to alarm again
next_periodic_time
Optional attribute; read/write access; type: f
Simulation time to set periodic interrupt flag bit again
next_update_time
Optional attribute; read/write access; type: f
Simulation time to update ram values again
time
Pseudo attribute; read/write access; type: s
Time of RTC in format "YY-MM-DD hh:mm:ss".
