ich10_usb_ehci

Description

USB EHCI host controller in Intel® ICH10.

Interfaces Implemented

conf_object, log_object, io_memory, pci_device, usb

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.

Port Objects

bank.pci_config
ich10_usb_ehci.pci_config
bank.usb_ctrl
ich10_usb_ehci.usb_ctrl
bank.usb_regs
ich10_usb_ehci.usb_regs

Commands for this class

  • info – print information about the object
  • print-descriptors – print descriptors
  • status – print status of the object

Attributes

async_list_polling_enabled
Optional attribute; read/write access; type: b
Poll the asynchronous list in 1ms intervals.
chipset_config
Required attribute; read/write access; type: o|[os]
Chipset configuration registers

Required interfaces: int_register.

companion_hc
Optional attribute; read/write access; type: [o|[os]|n{0:6}]
Per port companion host controller.

Required interfaces: usb.

config_registers
Pseudo attribute; read-only access; type: [i*]
The PCI configuration registers, each 32 bits in size.
expansion_rom_size
Optional attribute; read/write access; type: i
The size of the expansion ROM mapping.
pci_bus
Required attribute; read/write access; type: o|[os]
The PCI bus this device is connected to, implementing the pci-bus interface.

Required interfaces: io_memory, pci_bus.

pci_dev_num
Required attribute; read/write access; type: i
PCI device number on the ICH internal PCI bus
pcidev_func_num
Required attribute; read/write access; type: i
Function number in the PCI device
periodic_list_polling_enabled
Optional attribute; read/write access; type: b
Poll the periodic list in 0.125ms intervals.
