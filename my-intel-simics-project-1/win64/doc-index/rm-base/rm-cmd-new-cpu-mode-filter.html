new-cpu-mode-filter

Synopsis

new-cpu-mode-filter ["name"] [mode]

Description

Creates a new filter with the name given by the name argument. The filter restricts instrumentation to be enabled only when the processor executes in a certain mode(s). The mode argument can be used to configure the filter for a specific mode. The filter is added tool(s) with the tool specific <tool>.add-filter command.

Provided By

cpu-mode-filter

See Also

<cpu_mode_filter>.add-mode, <cpu_mode_filter>.remove-mode, <cpu_mode_filter>.delete
