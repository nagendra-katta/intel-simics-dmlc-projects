<service_node_comp>.ntp-virtual-time-mode

Synopsis

<service_node_comp>.ntp-virtual-time-mode

Description

Set virtual time mode to be used as base time over NTP. The current virtual time base will be used. To set virtual the time use 'ntp-set-virtual-time.
