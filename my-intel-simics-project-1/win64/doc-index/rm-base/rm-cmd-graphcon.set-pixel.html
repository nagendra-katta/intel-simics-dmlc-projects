<graphcon>.set-pixel

Synopsis

<graphcon>.set-pixel x y rgb

Description

Set pixel value at (x, y) from given pixel data rgb, either as a 3-element list with format [R, G, B], or as a hex string "#RRGGBB".

Provided By

graphcon
