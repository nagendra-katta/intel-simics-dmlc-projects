interface_log_tool

Description

Log interface calls

Interfaces Implemented

conf_object, log_object, instrumentation_tool

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.

Commands for this class

  • add-filter – add filter to the tool
  • control-blocked-interfaces – control blocked interfaces
  • control-logged-interfaces – control logged interfaces
  • delete – deletes instrumentation tool
  • disable-instrumentation – disable instrumentation
  • enable-instrumentation – enable instrumentation
  • info – print information about the object
  • list-providers – list-provider objects
  • remove-filter – remove filter from the tool
  • remove-instrumentation – remove instrumentation
  • status – print status of the object

Attributes

file
Pseudo attribute; read/write access; type: s|n
Write log to this file.
