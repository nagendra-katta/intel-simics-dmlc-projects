unsigned

Synopsis

unsigned int
unsigned16 int
unsigned32 int
unsigned64 int
unsigned8 int

Description

Interpret an integer, int, as an unsigned value of a specific bit width. For example unsigned16 -1 will return 0xffff. The unsigned command assumes a 64 bit width.
