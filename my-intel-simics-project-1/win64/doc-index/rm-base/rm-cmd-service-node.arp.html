<service-node>.arp

Synopsis

<service-node>.arp [-d] ["delete-ip"]

Description

Prints the ARP table for the service node if called with no arguments. An ARP table entry can be deleted by specifying the -d flag and the IP address in delete-ip.

Provided By

service-node
