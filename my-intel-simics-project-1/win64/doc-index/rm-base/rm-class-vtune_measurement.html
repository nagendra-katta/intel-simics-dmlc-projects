vtune_measurement

Description

Extension which runs VTune™ from Simics which connects to the current Simics session.

Interfaces Implemented

conf_object, log_object

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.

Commands for this class

  • flamegraph – produce a flamegraph (.svg) file of the profile
  • info – print information about the object
  • launch-gui – start vtune-gui on collected data
  • module-profile – show module profile of run
  • profile – show profile of run
  • save-folded-stacks – fold stacks of the profile and write it to a file
  • start – start measurement
  • status – print status of the object
  • stop – stop measurement
  • summary – show VTune summary of the run
  • thread-groups – show available thread groups
  • thread-profile – show thread profile of run

Attributes

vtune_path
Pseudo attribute; read/write access; type: s|n
path to vtune binaries, saved in prefs
