enable-vmp-feature

Synopsis

enable-vmp-feature "feature"

Description

Enable VMP simulation feature. Supported values for feature backoff, direct_rdtsc, ept, five_level_paging, nested_ept, shadow_vmcs, tpr_threshold, unrestricted_guest
