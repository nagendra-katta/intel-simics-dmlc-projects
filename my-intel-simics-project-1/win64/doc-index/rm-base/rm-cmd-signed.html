signed

Synopsis

signed int
signed16 int
signed32 int
signed64 int
signed8 int

Description

Interpret an integer, int, as a signed value of a specific bit width. For example signed16 0xffff will return -1. The signed command assumes a 64 bit width.

Provided By

Simics Core
