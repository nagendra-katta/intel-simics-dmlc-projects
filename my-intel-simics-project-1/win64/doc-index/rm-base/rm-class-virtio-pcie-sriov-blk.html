virtio-pcie-sriov-blk

Description

Virtio PCIE block device based on the Virtio 1.1 specification. Implements the base functionality along with SR-IOV support. Does not implement:
  • VIRTIO_BLK_F_SIZE_MAX
  • VIRTIO_BLK_F_SEG_MAX
  • VIRTIO_BLK_F_GEOMETRY
  • VIRTIO_BLK_F_RO
  • VIRTIO_BLK_F_BLK_SIZE
  • VIRTIO_BLK_F_FLUSH
  • VIRTIO_BLK_F_TOPOLOGY
  • VIRTIO_BLK_F_CONFIG_WCE
  • VIRTIO_BLK_F_DISCARD
  • VIRTIO_BLK_F_WRITE_Z
This device uses MSI-X for interrupts, and does currently not support legacy interrupts. CONFIG_PCI_MSI=y must thus be set in the kernel config.
