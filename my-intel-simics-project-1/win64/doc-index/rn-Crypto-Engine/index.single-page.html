Crypto Engine Release Notes

This document identifies important information for the Crypto Engine package for Simics 7. All users of this specific package should review this document carefully.

Changes Since Version 7.0.0

The following is a list of changes since the initial 7.0.0 release.

7.13.0 (build 7076)

7.12.0 (build 7064)

Cryptographic and secure hash library

  • Crypto-Engine converted to DML 1.4.

7.11.0 (build 7057)

Cryptographic and secure hash library

  • OpenSSL has been updated to version 3.4.0.

7.10.0 (build 7050)

Cryptographic and secure hash library

  • OpenSSL has been updated to version 3.3.2.
  • Fixed a problem where the SHA-160 HMAC could give incorrect results (fixes HSD-14023621597).

7.9.0 (build 7040)

Cryptographic and secure hash library

  • Added ECDSA Verify GF2 method (ecdsa_verify_gf2) on the ecdsa_v2 interface (fixes HSD-18039479212).

7.8.0 (build 7035)

Cryptographic and secure hash library

  • Added ECDSA Sign RS method for P256/P384 ECC (fixes HSD-18039635070).
  • Added ECDSA Sign GF2 methods (sign_r_gf2, sign_s_gf2, sign_rs_gf2) on the ecdsa_v2 interface (HSD-18039479103).
  • The ecdsa interface is deprecated, use the ecdsa_v2 interface instead.
  • Added method for generate RSA private key with reduced lambda on the rsa_v2 interface (fixes HSD-16025000866).

7.7.0 (build 7031)

Cryptographic and secure hash library

  • Fixed bug in pka-lib.c VT_Pka_Bn_Mod_Reduction.

7.6.0 (build 7029)

Cryptographic and secure hash library

  • Added RSA method encrypt_ex to support exponents bigger than 64 bits.
  • Added method for RSA key pair generation first form with reduced lambda.

7.5.0 (build 7027)

Cryptographic and secure hash library

  • Added LMS cryptographic algorithm based on Cisco open source implementation.
  • Interfaces compiled against DML 1.4
  • Added methods to generate r, s signs and prime number p for dsa_v2 interface.
  • The dsa interface is deprecated, use the dsa_v2 interface instead.

7.4.0 (build 7023)

Common

  • The signing certificate on Simics packages has been updated.

Cryptographic and secure hash library

  • Accurate license files are now included.

7.3.0 (build 7019)

Cryptographic and secure hash library

  • Add missing INTERFACES CMake mark-up for the crypto-engine module

7.2.0 (build 7009)

Cryptographic and secure hash library

  • OpenSSL has been updated to version 3.2.1.

7.1.0 (build 7007)

Common

  • Updated the batch (.bat) files in the bin folder so that they do not leak environment variables to the current cmd.exe command shell on Windows (fixes SIMICS-21906).

Cryptographic and secure hash library

  • OpenSSL has been updated to version 3.2.0.
  • Added method for CRT key generation on the rsa_v2 interface (fixes HSD-22019127853).

7.0.0 (build 7006)

Common

  • Simics, when launched on Linux, no longer sources the file $HOME/.simics-5.sh (fixes SIMICS-21148)
