Simics Python Release Notes

This document identifies important information for the Simics Python package for Simics 7. All users of this specific package should review this document carefully.

Changes Since Version 7.0.0

The following is a list of changes since the initial 7.0.0 release.

7.10.0 (build 7080)

Python

  • Added missing libncurses libraries to package 1033 (fixes HSD-15017625594).
  • The libexpat library used by Python has been upgraded to version 2.7.1.

7.9.0 (build 7075)

7.8.0 (build 7066)

Python

  • Fixed a problem on Windows, where the GUI framework could not be initialised.

7.7.0 (build 7061)

Python

  • The Python library is no longer distributed in the Simics-Base package, only in package 1033.

mini-python

  • The mini-python binary is no longer distributed in the Simics-Base package, only in package 1033. Its trampoline is still available in Simics-Base.

7.6.0 (build 7050)

mini-python

  • Fixed a problem on Windows when running mini-python directly from a Simics-Base installation without any command line arguments.

7.5.0 (build 7041)

Python

  • On Linux, the Python library is now also distributed under the name libpython3.10.so.

7.4.0 (build 7033)

Python

  • Some Python modules have been upgraded. The new versions are PyYAML 6.0.1, distro 1.9.0, flake8 7.1.0, mccabe 0.7.0, psutil 6.0.0, pycodestyle 2.12.0, pyflakes 3.2.0, typing_extensions 4.12.2.

7.3.0 (build 7028)

Python

  • The importlib_metadata and zipp Python modules are not distributed anymore.

7.2.0 (build 7023)

Common

  • The signing certificate on Simics packages has been updated.

7.1.0 (build 7019)

Python

  • The Simics embedded Python interpreter has been upgraded to version 3.10.14.

mini-python

  • Fixed a problem where mini-python failed to run on Windows machines with restrictive execution policies.

7.0.0 (build 7013)
