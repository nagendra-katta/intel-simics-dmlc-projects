Simics GDB Release Notes

This document identifies important information for the Simics GDB package for Simics 7. All users of this specific package should review this document carefully.

Changes Since Version 7.0.0

The following is a list of changes since the initial 7.0.0 release.

7.7.0 (build 7080)

GDB

  • The libexpat library used by GDB has been upgraded to version 2.7.1.

7.6.0 (build 7075)

7.5.0 (build 7071)

GDB

  • Fixed a problem starting GDB on Windows.

7.4.0 (build 7050)

GDB

  • The GDB version has been updated to 14.1.

7.3.0 (build 7029)

7.2.0 (build 7023)

Common

  • The signing certificate on Simics packages has been updated.

7.1.0 (build 7007)

Common

  • Updated the batch (.bat) files in the bin folder so that they do not leak environment variables to the current cmd.exe command shell on Windows (fixes SIMICS-21906).

7.0.0 (build 7006)

Common

  • Simics, when launched on Linux, no longer sources the file $HOME/.simics-5.sh (fixes SIMICS-21148)
