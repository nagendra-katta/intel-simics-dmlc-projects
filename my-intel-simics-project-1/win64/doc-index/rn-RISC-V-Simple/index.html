RISC-V Simple Release Notes

This document identifies important information for the RISC-V Simple package for Simics 7. All users of this specific package should review this document carefully.

Changes Since Version 7.0.0

The following is a list of changes since the initial 7.0.0 release.

7.10.0 (build 7075)

7.9.0 (build 7069)

Blueprints for a RISC-V simple platform

  • The RISC-V-Simple package now requires minimum Simics-Base 7.31.0.

RISC-V* Simple Target

  • Updated documentation to set fixed OpenSBI version v1.4. Later versions sets physical offset to 0x0 in ELF images while earlier sets them to 0x80000000. Currently the platform expects the ELF offset at 0x80000000 (fixes SIMICS-23006).

7.8.0 (build 7056)

RISC-V* Simple Board Component

  • No changes.

7.7.0 (build 7055)

RISC-V* Simple Target

  • Added a Goldfish RTC device which can be used to set the target time to the host time (the default) or a fixed time via the hardware:rtc:time target parameter.
  • Fixed a problem with target parameter default overrides (fixes SIMINT-1652).

7.6.0 (build 7035)

RISC-V* Simple Target

  • Make use of new allow_empty target parameter flag. The package now requires Simics-Base 7.19.0 or later.

7.5.0 (build 7027)

RISC-V* Simple Target

  • Fixed a problem when using writable persistent state. The package now requires Simics-Base version 7.14.0 or later.

7.4.0 (build 7023)

Common

  • The signing certificate on Simics packages has been updated.

7.3.0 (build 7017)

RISC-V* Simple Target

  • Fixed Linux multi target script to set unique default seeding values for the Virtio Entropy devices on the different machines (fixes SIMICS-22307).
  • The bootloader used is now the Dynamic Information Firmware from OpenSBI. The dynamic bootloader allows information about the next booting stage to be runtime configurable. Previously the Jump Address Firmware from OpenSBI was used, which instead had this information statically provided during compiliation (fixes SIMINT-1622).

7.2.0 (build 7016)

RISC-V* Simple Target

  • Fixed wrong status in DTS uart0 (fixes SIMINT-1620).
  • Fixed string stdout-path in the provided device tree config, risc-v-simple.dts in package (fixes SIMICS-22069).

7.1.0 (build 7007)

Common

  • Updated the batch (.bat) files in the bin folder so that they do not leak environment variables to the current cmd.exe command shell on Windows (fixes SIMICS-21906).
